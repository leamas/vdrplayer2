vdrplayer2 README
=================

Tool to replay logs created by OpenCPN Data Monitor in VDR mode.

Use `vdrplayer2 --help` for more info. On windows: `python vdrplayer2 --help`

Status: WIP, not much works. N0183 over TCP might, though

Installation:

    python -m build
    pip install dist/*.gz
